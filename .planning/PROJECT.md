# Project: NepalPrep

## Vision
A sophisticated medical entrance (CEE) preparation platform for Nepalese students.

## High-Level Goals
- Build a premium "Night Study" aesthetic interface.
- Implement an interactive and mobile-responsive dashboard.
- Transition from mock exams to consistent daily practice.
- Support 10,000–20,000 questions with lazy-loading performance.

## Tech Stack
- **Frontend**: React, Vite, Vanilla CSS.
- **Backend/Storage**: LocalStorage (auto-save), Firestore (progress tracking).
- **Hosting**: Vercel.

## Design Aesthetic
- Night Study (Modern, minimalist, dark-themed).
- Stethoscope & Heartbeat ECG animations.
- Premium HSL color palettes and micro-animations.
